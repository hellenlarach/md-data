Open Source Civic Data

Learn about and build applications with civic data for Miami

Apps

    Transportation

  • BusDrone - See the position of Miami busses in real time! Source on GitHub.
  • PDXLiveBus - See the position of Dade busses in real time! Source on GitHub
  • MetroCuts - Find bike racks and nearby businesses.
  • wherecanipark.in - Interpret Miami's Street Parking to determine where you can park. Source on Web App and API on GitHub

    Educational/How To

  • What Do I Do With...? - Allows residents and businesses in Miami-Dade County to find locations to recycle and responsibly dispose of unwanted items. Source on GitHub

    Fire/Crime

  • @miami911twitter bot - Source on GitHub

    Food safety/restaurant inspections

  • Dinegerous - Miami-Dade County Food Inspections

    Maps

  • Miami's Heritage Trees - Code on GitHub Source data
  • Map My Districts - Determine if a set of addresses are within a neighborhood or district boundaries by uploading bulk spreadsheets of Seattle addresses or drawing arbitrary polygons. Currently an internal tool for the Miami Chamber of Commerce but may open to the public in the future.

    Wikis

  • MiamiWiki - A community-authored resource for Miami, FL
